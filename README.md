# SOS Remédios

## Sobre
Aplicativo com o objetivo de conectar pacientess e farmácias em um lugar só, ao invés dos pacientes procurarem de farmácias em farmácias por um único remédio, o SOS Remédio centraliza todas essas informações em um único local, aumentando a velocidade no tratamento de doenças diminuindo a demora ao procurar remédios.

## Tecnologias utilizadas
- Java
- SpringBoot
- Flutter
- Dart
- PostgresSQL
- Postman
- Docker
